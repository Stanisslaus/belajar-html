Install and configure Apache On Ubuntu

1. Overview

Apache comes pre-installed on some Ubuntu Server images, but if it isn't already installed, you can install it with the following command:

sudo apt update
sudo apt install apache2

Apache is an open source web server that's available for Linux servers free of charge. In this tutorial we'll be going through the steps of setting up an Apache server.
